JDebug

JDebug is a collaborative JavaScript debugging tool. It give you an easy way to trace javascript info without any plugin. and it can auto syntax-hightlight you debug info. Just use on simple line Debug.trace() and pass you vars into and JDebug will show you on the page. JDebug is base on Dean Edwards Base lib,So before using JDebug,you wanna include Base.js and JDebuger.js,

<SCRIPT LANGUAGE="JavaScript" src="./js/Base.js"></SCRIPT>
<SCRIPT LANGUAGE="JavaScript" src="./js/JDebuger.js"></SCRIPT>

usage:

  1. using Debug.run=1 to enable debug,using Debug.run=0 to disable Debug window.
  2. using Debug.trace(Param) to log info.
